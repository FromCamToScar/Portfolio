The Stair Problem Solving Method

Home CSS Python Stair
S State The Problem
  • Describe the problem in Human language
  • Resist the urge to get technical here
  • Decide exactly what you intend to accomplish
  • how will you know when you have succeeded ?
  • This is the hardest step
T Find Your Tools
  • Might be ideas or processes
  • Could be research resources
  • Could be commands
  • Might be a program or software device
  • Might be a person
  • Don't limit yourself here
A Approach or Algorithm
  • What are the steps ?
  • How are you going to use the tool ?
  • What about reptition ?
  • What about constraints?
  • Develop a strategy
  • Restate the problem
  • Choose the most appropriate tools
I Implement
  • The Nike Step:"Just do it !"
  • Take each step,perform the related action
  • This is easy if the other steps have been taken
  • We usually try to start here but it should happen late in the process
R Refinement
  • It never works perfectly the first time
  • Expect issues-they are feedback
  • Issues are opportunties togather information
  • Test each step to find what caused issues
  • Refinement could mean create a diffrent approach !
